「指间灵动，快码加编」：阿里云通义灵码，再次降临博客园

博客园专属推广链接 https://tongyi.cnblogs.com/ （打开链接用阿里云账号登录，vscode 安装通义灵码插件完成问答互动）

第一期推广目标是7月底前邀请1万人

  • 截止6月21日，已邀请316人
  • 截止6月28日，已邀请370人

园子目前的主要收入来源是会员、周边、广告，在当前会员与周边收入很少的情况下，随着今年广告业务的回暖，广告收入成为维持生存的新希望。

虽然因为被百度降权失去了巨大的搜索流量，但如果找到长期合作的广告单子，基于园子高质量的用户群，依靠现有的流量，努力做好推广，通过广告收入维持基本生存是可行之路。

但残酷的现实是，绝大多数广告主都会直接找业内遥遥领先的流量霸主合作，我们很少有这样的机会，当机会出现时，我们格外珍惜。

今年1月就出现了这样的机会，阿里云首次在园子里尝试推广通义灵码（详见 通义灵码，降临博客园），看看推广效果，如果推广效果好就会考虑长期合作。为了争取后续的合作机会，当时我们花了很大力气在园子里推广，超额完成了推广任务，然后满怀希望地等待通义灵码的再次降临。

然而，这一等就是4个月，扛过了4个月的坎与难，终于在端午假期前与阿里云达成了通义灵码的 CPA(Cost Per Action) 推广合作，而且这是一个长期合作，合作到明年3月底，合作内容是帮助通义灵码拉新，每个月按照有效激活用户数付费。如果做到一定规模的用户数，对于困境中的园子将是一笔可观的收入。

终于等到这个有望加快园子脱困的好单子，虽然我们知道面向开发者人群做 CPA 广告是多么的不容易，但我们会想方设法在不影响用户体验的前提下解决这个推广难题。

「指间灵动，快码加编」，通义灵码在园子里的推广，随着这篇博文的发布而开启，期待大家的理解与支持。

如果您想体验AI编程助手，如果您找一款灵气的AI编程助手，如果您还没有用过通义灵码，期待您访问园子的专属推广链接 tongyi.cnblogs.com 并使用阿里云账号登录，然后安装通义灵码的 Visual Studio Code 或者 JetBrains IDEs 插件体验一下，比如向通义灵码问个代码问题。

注：使用阿里云账号登录后访问 tongyi.cnblogs.com，如果当前的阿里云账号之前没有与通义灵码账号关联过，会自动进行关联并计入园子带来的新用户，如果这位新用户使用了通义灵码（与通义灵码问答互动），就算一个有效用户。

具体操作步骤如下：
1）浏览器打开 https://tongyi.cnblogs.com/ 会自动跳转到园子的专属推广链接（域名是 developer.aliyun.com）
2）等页面出现 活动提示 弹出框
3）如果您还没登录阿里云官网，弹出框中会出现 登陆/注册通义灵码 按钮，点击按钮完成登录
4）如果您已登录阿里云官网，弹出框中会出现 立即下载体验 按钮

5）点击 立即下载体验 按钮，进入通义灵码安装页面（域名是 tongyi.aliyun.com）
6）点击页面中的 立即安装 按钮，唤起 Visual Studio Code 插件市场安装，或者自己在插件市场搜索 Lingma 并安装

7）如果您使用的是 JetBrains IDEs，可以到对应的插件市场搜索 TONGYI Lingma 进行安装
8）安装完成之后，使用通义灵码至少进行一次交互操作，比如问个问题
注：关键是第4、6或7、8步

期待您的支持！欢迎访问园子的专属推广链接，等页面出现「活动提示」后，点击按钮用阿里云账号登录，然后安装通义灵码体验一下（与通义灵码问答互动）。
